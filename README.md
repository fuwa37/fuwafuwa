# Deskripsi
Web service menunjukkan lokasi kantin-kantin yang berada dalam ITB dengan menu-menu yang uang disediakan masing-masing kantin

# Lingkup operasi
...
- DBMS   :MySQL
- Bahasa :Go
  - Framework  :Beego
...
Aplikasi dapat dibuka melalui web browser
